## 🌍 What is KBTC?

**Kongali Bitcoin (KBTC)** is an ERC-20 asset designed to reflect the scarcity principles of Bitcoin while leveraging the flexibility and interoperability of Ethereum.

KBTC is designed as:

- 🏦 A symbolic digital reserve  
- 🔗 A future DeFi & cross-chain asset  
- 👥 A community-driven token for Web3 development  
- 🌐 A base token for the Kongali Ecosystem  

Currently, KBTC is in an **early on-chain deployment phase** — the contract is live, verifiable, and immutable.

KBTC mirrors Bitcoin’s supply to emphasize:
